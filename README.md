# Memory Match Game

**College WPF project (2022)** - flip cards, match animal pairs, beat timer.

## 🎮 How to Play
- **PLAY** - shuffle cards
- Click 2 cards - find matches  
- Matches disappear
- Clear all 12 pairs!
